# Subtitle-Extraction-and-Visualisation

To use 'Option 1: Transcribe Media Folder', you must:
1) Run 'run.bat', which is a setup script that creates the virtual environment, installs dependencies, and launches the app automatically.
2) Run 'server.py' and keep it running in the background
3) Paste the folder path in which your file to be transcribed is (note - it must be a folder, not an individual file. Working on fixing this)
4) Paste your unique Hugging Face token (if you're unsure how to get this, follow the instructions in [A Guide to Extracting Subtitles.pdf](https://github.com/user-attachments/files/29846177/A.Guide.to.Extracting.Subtitles.pdf)
5) You will also need to have FFmpeg and WhisperX installed, as detailed in the document above.
