# Subtitle-Extraction-and-Visualisation

run.bat is a setup script that creates the virtual environment, installs dependencies, and launches the app automatically.
You will also need to have FFmpeg and WhisperX installed, as detailed in this document - 
[A Guide to Extracting Subtitles.pdf](https://github.com/user-attachments/files/29846177/A.Guide.to.Extracting.Subtitles.pdf)
