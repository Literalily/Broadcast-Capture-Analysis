Option 1: Transcribe Media Folder

Use WhisperX to transcribe a video/audio file.

To do this, you must:
1. Paste the file path of the desired folder to be transcribed, (e.g. "D:\Broadcast data\radio1\NewsAtOne")
Compatible file types: mp4 mkv mov avi m4v ts mp3 wav m4a flac
2. Paste your unique Hugging Face token. If you're not sure how to do this, follow the instructions in the README file.
